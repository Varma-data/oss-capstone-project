Open Source Audit for MySQL

Software Type

MySQL is an open source relational database management system (RDBMS) designed for storing and managing data. It is commonly used in web application development and is also part of the LAMP stack.

Included Scripts
1. System Identity Report

A script that shows basic information about the system such as OS name, kernel version, user, uptime and date.

2. FOSS Package Inspector

A script to check to see if MySQL is installed and give some basic details for the package.

3. Disk and Permissions Auditor

A script to audit important directories and display the permission and size of them. It will also verify that the MySQL configuration directory is correct.

4. Log file analyzer

A script to read a log file and display the number of occurrences of keywords (e.g. "error").

5. Open Source Manifesto Generator

A script that prompts the user for input and generates a small open source statement, which gets saved as a text file.
